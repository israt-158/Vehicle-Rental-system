# Vehicle Rental System

## Overview

Vehicle Rental System is a console-based application developed in C Programming. It helps manage vehicles, customers, bookings, returns, and billing operations efficiently.

## Features

* Add and Remove Vehicles
* Search Vehicle by ID
* Search Vehicle by Type
* Display All Vehicles
* Book Vehicle
* Return Vehicle
* Customer Management
* Booking History
* Generate Bill
* File Handling for Data Storage

## Data Structures Used

* Array
* Linked List
* Queue
* Stack

## Technologies

* C Programming
* File Handling
* Dynamic Memory Allocation

## How to Run

1. Compile the code:
   gcc vehicle_rental.c -o vehicle_rental

2. Run the program:
   ./vehicle_rental
